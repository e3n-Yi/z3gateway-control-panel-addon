# 更新日志

## 0.1.14

- **过零校准 · 双向通过判定**：通过范围现在同时判定半波两端。例如窗口设 500us 时，测量值 ≤500us 或 ≥9500us（10ms 半波的另一端）都算达标，避免「只认滞后、一直只超前一点点却补不出」导致反复补偿卡死。
- **自动过零校准面板可折叠**：并非所有设备都需要该功能，默认折叠；校准启动时自动展开并滚动到顶部。
- **修复面板滚动**：桌面布局 / 浏览器放大时，自动过零校准面板会被压缩、下半部分（关灯指标、原始串口、开始/停止按钮）滚不出来——现已修复。
- 内部：将构建时补丁/脚本固化进源文件，简化发布流程（对功能无影响）。

## 0.1.6 – 0.1.13

- 自动过零校准面板及其多轮调参（发送间隔、通过范围、指标显示、仪器原始串口面板等）。
