# Z3Gateway Control Panel Add-on Repository

Home Assistant custom repository for the Z3Gateway Control Panel app.

## Install

Add this repository URL in Home Assistant App Store, then install `Z3Gateway Control Panel`.

Current package supports `aarch64` only.

## Notes

- Use a dedicated Silicon Labs EZSP/NCP dongle.
- Do not share the same dongle with ZHA or Zigbee2MQTT.
- The packaged gateway host binary is built for ARM64/aarch64.
